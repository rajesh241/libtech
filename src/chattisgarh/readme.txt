This directory has all the code for the Libtech Chattisgarh Project.

Following are the directories.

fto: Has all the code related to processing and crawling ftos
muster: has all the code related to processing and crawling musters
jobcardregister: has all the code for jobcards
db: has the database schema. This does not have the DB, but just database schema. You can populate the database by running the crawl scripts.
misc: This has misc codes
