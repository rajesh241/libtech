Libtech Broadcast Management System

Instructions for Creating and Managing Broadcast Related Links

Following are the Steps that are required to do the Broadcast

  1. Determine the Type of Broadcast
    There are two types of Broadcasts.
    • Group Based This Broadcast will go to predefined groups. Like Panchayat president groups, karyakartas group. If you have set of existing numbers then you can choose this type of Broadcasts.
    • Geo Location Specific This Broadcast will go to people specific to a particular location. you can Broadcast to entire district, block or panchayat.
  2. Create a Group
    For the first time when you want to Broadcast to a specific group. You would first need to create a Group. You can create a group and add numbers to it. The needs to be done only once. (Note the system also allows you to add numbers to existing groups)
  3. Upload audio file
    You need to upload the audio file. Do note that the audio file needs to be there in only .wav format. Other audio files at present are not accepted. If you have an audio file which is not in .wav format, you can covert it by the following two methods
    • Do it through the audacity software.
    • Do it online by going to media.io
    Once you will upload the audio file you will get the file ID from both the places. You need to fill the fileid in the next step`
  4. Fill the create Broadcast Form
    Once you have done the above you can create the broadcast, by filling the form. You would have to fill various details, such as startDate, endDate and various other parameters of Broadcast.
  5. Place a test Call to your phone number
    Follow the approve broadcast link and place a test call to your number. If you get a test call and it is satisfactory approve the Broadcast.
  6. Check Broadcast Reports
    Once approved, now you can periodically check the Broadcast Report to ensure that your calls are going. Once the broadcast is finished you can download the reports and analyze them.
